# AI Internship Assignment 

This repository contains solutions for the AI Internship Assignment.

## Tasks Covered
- Task 1: AI-Based Automated News Video Generation
- Task 2: SEO Blog Post Creation Tool
- Task 3: High-Level to Low-Level Architecture Pipeline

Each task includes:
- Source code
- Outputs
- Detailed report (PDF)

## Tech Stack
- Python
- Google Gemini API
- BeautifulSoup
- MoviePy
- PIL

## Video Outputs
The video files are included in the repository under Task 1 / outputs.  
If GitHub cannot preview the MP4 files in browser, please download them locally to play.


## How to Run
Each task folder contains a `code` directory with a `main.py` file.
Install dependencies using `requirements.txt` and run `main.py`.
